This plugin changes the widgets in the dashboard to a horizontal line.

== Description ==

Dashboard widgets are arranged vertically by default, in several columns. This plugin changes the widgets to horizontal, aligning the height of adjacent widgets for a cleaner look.

Widgets will also be less restricted in width, allowing content to be displayed that could not previously be represented.

= Some features: =

* It changes widgets that are normally aligned vertically to horizontal.
* The width of each widget can be increased or decreased relative to each other.
* Enhances the appearance of the placeholder to which you are dragging.
* Some settings can be changed in the `jp12net\align_widgets_horizontally\settings` filter.

== Installation ==

1. A plug-in installation screen is displayed on the WordPress admin panel.
2. It installs it in `wp-content/plugins`.
3. The plug-in is made effective.
4. Go to the dashboard and drag and drop widgets as you place them.
5. If you want to change the width of the widget, click on `Screen Options` and adjust it using the number input box next to the widget name.

== Frequently Asked Questions ==

= Are widget placements saved by user? =

Yes, the placement and width of each widget is saved per user.
